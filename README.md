# 🚀 NovaTab – Smart New Tab Launcher

Turn your new tab into a powerful productivity hub.

---

## ✨ Features

- 🔗 Unlimited custom shortcuts  
- 🌙 Dark mode (System / Light / Dark)  
- ⚡ Fast and minimal UI  
- 🧠 Smart favicon loading (Google + DuckDuckGo fallback)  
- 🔄 Cross-tab real-time sync  
- 🔒 One-tab-per-session mode  
- ⌨️ Keyboard shortcuts (Ctrl+K, Ctrl+N)  
- 📦 Import / Export shortcuts  
- 🛡 Fully CSP-compliant

## 🛠 Installation

1. Download or clone this repository  
2. Go to `chrome://extensions/`  
3. Enable **Developer Mode**  
4. Click **Load unpacked**  
5. Select the extension folder
